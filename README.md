# Moba Manager / LoL Manager

Simulador de gestão de esports *League of Legends* focado no CBLOL 2026.

## Como rodar (Windows)

```bat
run_game.bat
```

## Stack

- Backend: FastAPI + SQLAlchemy async + SQLite (dev)
- Frontend: React + Vite + Tailwind
- Testes: `pytest tests -q` com `PYTHONPATH=.`

Ver `CONTINUIDADE.md` e `docs/HANDOFF_SESSAO.md` para continuidade de sessão.
